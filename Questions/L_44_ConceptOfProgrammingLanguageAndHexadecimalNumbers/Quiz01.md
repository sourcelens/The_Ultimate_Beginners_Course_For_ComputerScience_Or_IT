1 : Which are examples of Programming languages?  

a) C++  
b) Java

2 : 0x is the prefix written before a _______  

a) Decimal number  
b) Hexadecimal number  
c) Octadecimal number  
d) Binary number  

**Answer** b) 

**Description**  

Ox is written before a Hexadecimal number as a prefix.
